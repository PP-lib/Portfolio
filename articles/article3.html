Disqusを利用したコメント欄の導入と設定

概要

本記事では、GitHub PagesでホストしているサイトにDisqusを使ってコメント欄を導入する方法について解説します。 Disqusはウェブサイトのコメント管理に便利なプラットフォームで、基本的な機能は無料で利用できます。

今日の作業内容

  • Disqusの導入準備
    • Disqusアカウントの作成
    • サイトの登録とshortnameの取得
  • HTMLへの組み込み
    • コメント欄を表示するためのコードを各記事ページに挿入。
    • 短縮名（shortname）を使ってDisqusを動作させる設定を行いました。
  • CSSでデザインを調整
    • コメント欄のレイアウトやスタイルを整え、サイトのデザインに統一感を持たせました。
  • 公開と動作確認
    • GitHub Pagesに公開し、ブラウザ上でコメント欄が正しく表示されることを確認しました。

手順：Disqusを使ったコメント欄の追加

Disqusの設定方法に従い、短縮名を正しく使用してください。
