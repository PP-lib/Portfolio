6 Dec24: GitHub Pages 構築記録

ホームに戻る

記事内容

GitHub Pages を使ったウェブサイト構築の記録です。

構築手順

  1. GitHub アカウントを作成
  2. ローカルでHTML、CSSファイルを作成
  3. リポジトリを作成し、ファイルをアップロード
  4. GitHub Pages を設定してサイトを公開

重要なポイント

  • Personal Access Token を使用して認証
  • ローカルサーバーで動作確認
  • Git コマンドを使用した効率的な管理
